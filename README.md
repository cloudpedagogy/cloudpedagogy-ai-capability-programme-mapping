# AI Capability Programme Mapping

A lightweight tool for estimating AI capability coverage and exposure across a full academic programme or curriculum structure.

## 🔗 Role in the CloudPedagogy Ecosystem

**Phase:** Phase 3 — Capability System

**Role:**
Estimates and maps AI capability coverage across curriculum structures (programmes and modules) to identify exposure and alignment gaps.

**Upstream Inputs:**
Curriculum structural data from the **Mapping Engine** and baseline capability benchmarks from the **Capability Assessment Tool**.

**Downstream Outputs:**
Provides capability-aware curriculum insights for the **Curriculum Simulation Tool** and institutional **Governance Dashboard**.

**Does NOT:**
- Perform detailed workload or assessment simulation.
- Record live individual human-AI decision outcomes.

For a full system overview, see: [SYSTEM_OVERVIEW.md](../SYSTEM_OVERVIEW.md)

---

## Overview

The **AI Capability Programme Mapping** tool bridge the gap between abstract capability frameworks and concrete curriculum implementation. It allows academic leads to visualize which parts of a degree programme explicitly foster AI capability and where exposure to AI-driven risk is highest.

## Key Features

- **Mapping Layer**: Align specific modules to the six CloudPedagogy AI Capability domains.
- **Exposure Estimation**: Calculate the cumulative "AI Exposure" a student receives throughout a programme.
- **Alignment Signals**: Flag modules that lack explicit capability development despite having high AI involvement.

## Disclaimer

This repository contains exploratory, framework-aligned tools developed for reflection, learning, and discussion.

These tools are provided **as-is** and are not production systems, audits, or compliance instruments. Outputs are indicative only and should be interpreted in context using professional judgement.

All applications are designed to run locally in the browser. No user data is collected, stored, or transmitted.

All example data and structures are synthetic and do not represent any real institution, programme, or curriculum.
